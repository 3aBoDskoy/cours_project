Здесь используем подключение по ssh, подключение такого типа используется для удаленной работы 
на серверах.
Команды для терминала:

ssh-keygen # генерируем собственный секретный и публичный ключи
cd .ssh  # заходим в папку с ключами
nano id_rsa.pub # открываем его редактором обязательно pub!! - публичный
перекидываете мне в телеграмм
